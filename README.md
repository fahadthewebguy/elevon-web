# Elevon Web

Landing page for **Elevon Web** — fast, high-converting website design & development for U.S. businesses.

## About

A single-file, self-contained marketing landing page (`index.html`) with:

- Responsive, mobile-first layout
- Animated hero, before/after results, an interactive drag comparison, pricing/offer, testimonials, FAQ
- Self-contained SVG/CSS visuals — no external image dependencies
- All lead capture routed to **WhatsApp** (no backend or form processing required)

## Run locally

It's a static page — just open `index.html` in a browser. Or serve it:

```bash
npx serve .
```

## Deploy

Because it's a single static file, it can be hosted anywhere: GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static host.

To enable **GitHub Pages**: repository *Settings → Pages → Deploy from a branch → `main` / root*.
